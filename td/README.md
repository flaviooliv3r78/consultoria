# TD Consultoria

Landing page profissional para a TD Consultoria, criada para apresentar os serviços, expertise e formas de contato da empresa no segmento de laminação de embalagens flexíveis.

## Objetivo

Transformar a presença digital da empresa em uma experiência mais profissional, clara e persuasiva, com foco em:
- maior credibilidade institucional;
- melhor experiência do usuário em desktop e mobile;
- comunicação objetiva dos serviços e diferenciais;
- incentivo a contato via WhatsApp e e-mail.

## O que foi aplicado

### UX / UI
- layout limpo e visualmente profissional;
- hierarquia de conteúdo mais clara;
- botões com foco em conversão;
- melhor espaçamento, contraste e consistência visual;
- navegação mais intuitiva e acessível.

### Usabilidade
- menu simples e direto;
- navegação por âncoras com rolagem suave;
- CTA principal claro para agendamento;
- botão flutuante de WhatsApp para contato rápido;
- experiência responsiva para diferentes telas.

### Regras de negócio aplicadas
- contato principal deve ser simples e direto;
- mensagens de CTA devem incentivar ação imediata;
- informações de contato devem estar sempre acessíveis;
- conteúdo deve reforçar autoridade técnica e confiança;
- a comunicação deve priorizar benefício para o cliente, não apenas características do serviço.

## Como visualizar localmente

1. Abra a pasta do projeto no navegador.
2. Execute o arquivo index.html diretamente ou use um servidor local simples.

Exemplo com Python:

```bash
python -m http.server 8000
```

Depois acesse:

```text
http://localhost:8000
```

## Próximos passos recomendados
